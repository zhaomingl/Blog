# 箭头函数

在定义函数的时候，除了可以使用前面的声明函数和函数表达式之外，我们这里还可以使用箭头函数。

**其实箭头函数可以理解为函数表达式的简写模式。**

例如：

*函数表达式：*

    let sum = funcition(a, b) {
        return a + b;
    }

*箭头函数：*

    let sum = (a,b) => {
        return a + b;
    }

    或者：
    let sum = (a,b) => a + b;

**注意：**

- 箭头函数的参数只能有一个，如果有多个，需要使用小括号括起来。
- 箭头函数如果没有参数时，括号里面是空的是，但是不能省略括号。

**多行的箭头函数**

箭头函数不仅仅可以写一行代码，但是书写多行代码的时候，我们需要使用花括号是将代码块包含起来。

    let sum = (a,b) =>{
        let c = a + b;
        return c;
    };
    console.log(sum(1,2));
